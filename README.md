# Hotel Booking Website

A responsive hotel booking landing page built with Vite, React, and Tailwind CSS. It showcases destination highlights, weekend deals, and curated sections to guide users from discovery to booking.

## Features
- Modern, responsive UI
- Section-based layout (hero, trending destinations, testimonials, deals)
- Reusable React components
- Tailwind CSS styling

## Tech Stack
- Vite
- React
- Tailwind CSS

## Getting Started
1. Install dependencies:
	- npm install
2. Start the dev server:
	- npm run dev
3. Build for production:
	- npm run build

## Project Structure
- src/components: Reusable UI components
- src/layout: Page layout pieces (navbar, footer, etc.)
- src/sections: Page sections for the home view
- src/styles: Global styles
- public/assets: Images and static assets

## Scripts
- npm run dev: Start the development server
- npm run build: Build for production
- npm run preview: Preview the production build

## Notes
Assets are stored under public/assets and referenced by components and sections.
